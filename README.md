# 🎹 Piano App

A virtual piano built with Python and tkinter from scratch.

## Features
- 14 white keys and 10 black keys
- Click to play notes
- Sound feedback using winsound
- Color response on key press

## How to Run
1. Make sure Python is installed
2. Run: python piano.py

## Built With
- Python
- tkinter
- winsound
- threading
